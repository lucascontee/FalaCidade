# FalaCidade
O FalaCidade é uma aplicação que permite aos cidadãos registrar problemas urbanos, como buracos, ruas sem iluminação e postes danificados. As solicitações são feitas com fotos e a localização do problema. A solução agiliza a comunicação com a prefeitura e otimiza a gestão pública, com atualização de status em tempo real.
